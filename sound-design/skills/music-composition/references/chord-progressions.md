# Chord Progressions Database

Extended chord progression library organized by emotional character.

## Triumphant/Heroic

```
I - IV - V - I         (C - F - G - C)        Classic resolution
I - V - vi - IV        (C - G - Am - F)       Pop/anthem
I - IV - I - V         (C - F - C - G)        Simple triumph
I - iii - IV - V       (C - Em - F - G)       Building heroic
I - V - IV - V         (C - G - F - G)        Stadium rock
```

## Mysterious

```
i - bVII - bVI - bVII  (Am - G - F - G)       Classic mystery
Isus4 - I - IVsus2     (Csus4 - C - Fsus2)    Suspended wonder
i - III - bVII - IV    (Am - C - G - D)       Modal ambiguity
i - bVII - i           (Am - G - Am)          Simple question
```
*Tip: Modal vamps on one chord with color tones work well for mystery.*

## Modulation Techniques

| Technique | Movement | Effect |
|-----------|----------|--------|
| Relative | Am → C | Brightening |
| Parallel | C → Cm | Darkening |
| Up half-step | C → Db | Energy boost |
| Up whole-step | C → D | Key lift |
| Circle of 5ths | C → G → D | Natural progression |

## Cadences

| Type | Pattern | Use For |
|------|---------|---------|
| Perfect | V → I | Strong resolution |
| Plagal | IV → I | "Amen" resolution |
| Deceptive | V → vi | Surprise, extend phrase |
| Half | I → V | Pause, question |
