# Mood Parameters

Musical parameter mappings for each mood/emotional character.

## Parameter Overview

Each mood translates to specific musical parameters:

| Parameter | Range | Effect |
|-----------|-------|--------|
| Key | Major/Minor/Modal | Emotional foundation |
| Tempo | BPM | Energy level |
| Dynamics | ppp-fff | Intensity |
| Rhythm | Pattern | Movement feel |
| Harmony | Chord types | Tension/resolution |
| Texture | Density | Fullness |

---

## Mood Definitions

### Triumphant

**Emotional Character:** Victory, achievement, celebration

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Key | Major (C, G, D, F) | Positive, resolved |
| Mode | Ionian, Lydian | Bright, lifted |
| Tempo | 100-140 BPM | Energetic but stately |
| Dynamics | mf → fff | Building to powerful |
| Rhythm | Marcato, bold | Confident, proud |
| Harmony | I-IV-V, I-vi-IV-V | Resolved, consonant |
| Intervals | Perfect 4ths, 5ths | Fanfare quality |
| Texture | Full, layered | Grand |

**Instrument Suggestions:**
- Primary: Brass fanfare, full strings
- Accent: Timpani, cymbals
- Texture: Choir "aah"

---

### Tense

**Emotional Character:** Anxiety, uncertainty, impending danger

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Key | Minor (Am, Dm, Em) | Unsettled |
| Mode | Aeolian, Phrygian | Dark, exotic |
| Tempo | 80-120 BPM | Moderate, pressing |
| Dynamics | pp → ff | Building, unpredictable |
| Rhythm | Ostinato, syncopated | Relentless, unstable |
| Harmony | Diminished, sus4 | Unresolved |
| Intervals | Minor 2nds, tritones | Dissonant |
| Texture | Sparse → dense | Building |

**Instrument Suggestions:**
- Primary: Tremolo strings, low brass
- Accent: Staccato stabs
- Texture: Low drones, heartbeat pulse

## Mood Combinations

Moods can be layered for nuance:

| Combination | Result | Example Use |
|-------------|--------|-------------|
| Tense + Mysterious | Foreboding | Pre-boss buildup |
| Epic + Melancholic | Bittersweet victory | Pyrrhic victory |
| Playful + Mysterious | Whimsical wonder | Fairy exploration |
| Aggressive + Triumphant | Battle victory | Combat winning |
| Peaceful + Nostalgic | Warm memory | Flashback |
| Eerie + Tense | Horror buildup | Horror approach |

**Implementation:** Use primary mood for melody/harmony, secondary for texture/rhythm.
