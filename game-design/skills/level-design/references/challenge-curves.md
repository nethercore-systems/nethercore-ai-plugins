# Challenge Curve Templates

Ready-to-use difficulty pacing patterns.

---

## Standard Curve

```
Difficulty
100% │           ╱╲
 75% │         ╱    ╲
 50% │    ╱──╱        ╲
 25% │  ╱              ╲__
  0% │╱
     └─────────────────────→
       Intro  Mid  Peak  End
```

**Pattern:** Gradual rise → Peak before end → Slight drop for satisfying finish

## Challenge Beat Timing

| Level Length | Beat Count | Peak Location |
|--------------|------------|---------------|
| 5 min | 3-4 beats | 80% through |
| 15 min | 5-6 beats | 75% through |
| 30 min | 7-10 beats | 70% through |
| 60+ min | 12+ beats | Multiple peaks |

---

## Rest Points

Insert rest after:
- [ ] Major challenge completion
- [ ] New mechanic introduction
- [ ] Story/narrative beat
- [ ] Boss defeat
- [ ] Before final challenge

Rest duration: 10-20% of previous challenge duration

---

## Challenge Escalation Methods

| Method | Description |
|--------|-------------|
| **More enemies** | Quantity increase |
| **Harder enemies** | Quality increase |
| **Less resources** | Scarcity increase |
| **Time pressure** | Urgency increase |
| **Combination** | Multiple mechanics at once |
| **Environment** | Hazards, limited space |
| **Stakes** | Consequences of failure |
