# Engagement Patterns

---

## Session Structure

```
Hook → Rising engagement → Peak → Cool down → Cliffhanger
 5%         30%             40%      20%          5%
```

---

## Session Hooks

| Hook | Implementation | Strength |
|------|----------------|----------|
| Incomplete goal | Quest in progress | Strong |
| New unlock | Just received new ability | Strong |
| Mystery | Unanswered question | Medium |
| Resources at risk | Carrying valuable loot | Medium |
| Timer | Event ending soon | Variable |

---

## Return Motivators

### Short-term (next session)
- Cliffhanger ending
- Incomplete objective
- New content promised
- Social commitment

### Long-term (over weeks)
- Meta-progression
- Seasonal content
- Community engagement
- Mastery goals

---

## Engagement Killers

| Killer | Impact | Solution |
|--------|--------|----------|
| Excessive grind | Fatigue | Reduce requirements |
| No clear goal | Directionless | Better objective display |
| Punishing failure | Frustration | Softer death penalty |
| Repetitive content | Boredom | Add variety |
| Long load times | Breaks flow | Technical optimization |

---

## Engagement Metrics

Track these to measure engagement:

| Metric | Healthy Sign | Warning Sign |
|--------|--------------|--------------|
| Session length | Matches target | Too short / Too long |
| Return rate | 50%+ same-day | <20% same-day |
| Completion rate | 30%+ finish game | <10% finish |
| Drop-off points | Even distribution | Spike at specific point |

---

## Quick Engagement Test

After each playtest, ask:
1. Did you want to keep playing? (Y/N)
2. Will you play again? (Y/N)
3. When did you get bored? (timestamp)
4. When were you most engaged? (timestamp)
