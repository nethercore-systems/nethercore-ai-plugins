## Difficulty Etiquette

### Do
- Let players change difficulty anytime
- No shaming for easy mode
- Trophy/achievement parity when possible

### Don't
- Lock content behind difficulty
- Hide easy mode
- Mock players for choices
- Force restarts to change

---

## Implementation Notes

- Save difficulty per profile
- Allow changes mid-game
- Separate accessibility from difficulty
- Track what options players use
