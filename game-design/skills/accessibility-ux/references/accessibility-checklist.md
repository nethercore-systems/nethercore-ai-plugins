# Accessibility Audit Checklist

---

## Controls (Must Have)
- [ ] All controls remappable
- [ ] Control scheme presets
- [ ] Button prompts match controller
- [ ] No mandatory motion controls
- [ ] Settings saved between sessions

## Controls (Should Have)
- [ ] One-handed schemes
- [ ] Hold/toggle options
- [ ] No simultaneous button combos required
- [ ] Adjustable stick sensitivity
- [ ] Deadzone customization

---

## Visual (Must Have)
- [ ] Text size options
- [ ] Subtitles available
- [ ] High contrast mode
- [ ] UI scaling

## Visual (Should Have)
- [ ] Colorblind modes (3 types)
- [ ] Speaker labels on subtitles
- [ ] Caption backgrounds
- [ ] Screen shake toggle
- [ ] Flash reduction option

---

## Audio (Must Have)
- [ ] Separate volume sliders
- [ ] Subtitles for dialogue
- [ ] Mute option

## Audio (Should Have)
- [ ] Visual cues for audio
- [ ] Mono audio option
- [ ] Sound direction indicators
- [ ] Full closed captions

---

## Difficulty (Must Have)
- [ ] Multiple difficulty levels
- [ ] Difficulty changeable anytime
- [ ] No content locked by difficulty

## Difficulty (Should Have)
- [ ] Granular difficulty sliders
- [ ] Assist modes
- [ ] Skip options for stuck points
- [ ] Practice modes

---

## General (Must Have)
- [ ] Pause anywhere
- [ ] Clear tutorials
- [ ] Save frequently
- [ ] Responsive menus

## General (Should Have)
- [ ] Multiple save slots
- [ ] Session length estimates
- [ ] Break reminders option
- [ ] Comprehensive settings menu

---

## Testing

- [ ] Tested with colorblind simulation
- [ ] Tested with screen reader
- [ ] Tested one-handed
- [ ] Tested at minimum text size
- [ ] Tested without audio
