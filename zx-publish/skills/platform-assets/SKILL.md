# Nethercore ZX Platform Assets

## Required Assets

### Game Icon

| Property | Value |
|----------|-------|
| Size | 64x64 pixels |
| Format | PNG (RGBA) |
| Max Size | 50KB |

**Tips:**
- Bold, simple shapes visible at small size
- High contrast on dark background
- No text (illegible at 64px)
- Center the focal point

### Screenshots (Recommended)

| Property | Value |
|----------|-------|
| Size | 960x540 or 1920x1080 |
| Format | PNG or JPG (90+) |
| Count | Up to 5 |

**What to capture:**
1. Hero shot (best visual moment)
2. Core gameplay
3. Variety (different areas/modes)
4. Unique features

### Banner (Optional)

| Property | Value |
|----------|-------|
| Size | 1280x720 pixels |
| Format | PNG or JPG |

Used for featured games section.
