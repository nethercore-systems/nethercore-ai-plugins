# Agent Continuation Pattern

After completing work, include this prompt:

---
**Next Steps:**
1. [Most logical follow-up] → [agent/skill name]
2. [Alternative direction] → [agent/skill name]

Continue with #1? (yes/no/other)
---

## Common Continuations

| After | Suggested Next |
|-------|----------------|
| Scaffolding | Implement game logic, generate assets |
| Feature implementation | Add polish (sounds, particles), test feature |
| Asset generation | Integration, quality review |
| Code review | Fix identified issues |
| Build success | Run sync test, playtest |
